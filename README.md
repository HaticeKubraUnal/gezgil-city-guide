# 🌍 GEZGİL - City Guide Web Application

## 👉 [(Live Demo)](https://haticekubraunal.github.io/gezgil-city-guide/HTML/Gezgil.html)

A dynamic and fully responsive city guide website project introducing historical and natural places. This project was developed as part of the **Internet Technologies** course in **May 2025**.

## 🚀 About The Project

Gezgil is designed to guide users through the natural beauties, historical sites, and city attractions of the region. The project is built with plain HTML, CSS, and JavaScript, focusing on user experience, responsive design, and interactivity without relying on external frameworks.

## ✨ Key Features

* **Bilingual Support (TR/EN):** The entire platform offers seamless dual-language support, allowing users to switch between Turkish and English interfaces easily.
* **Interactive Search Engine:** Users can search for specific locations and be redirected to the relevant sections instantly.
* **Text-to-Speech (Accessibility):** Includes a JavaScript-powered voice reading feature for better accessibility.
* **Font Size Control:** Users can dynamically increase or decrease the text size for better readability.
* **User Authentication Simulation:** Features a login and registration system using browser `localStorage`.
* **Responsive Design:** Fully mobile-compatible layouts using Flexbox and CSS media queries.
* **Dynamic Media:** Includes animated video backgrounds and horizontal scrolling image galleries.

## 🛠️ Built With

* **HTML5** (Semantic structure)
* **CSS3** (Flexbox, custom animations, responsive media queries)
* **Vanilla JavaScript** (DOM manipulation, LocalStorage, SpeechSynthesis API)
